Privacy Policy (English)

Effective Date: 2024.05.22

1. Purpose of Personal Information Processing

‘nBit’ processes personal information for the purpose of providing services (app execution/termination) and does not use it for any other purposes.

2. Sensitive Permissions (Accessibility Service)

[Accessibility Permission]: This app requires accessibility permissions to detect the currently running app and perform automatic start/termination functions.

  • Data Access: Detects package names of launched apps.
  • Usage: Only used to trigger automation features set by the user.
  • Storage: This data is processed in real-time, is never stored on the device or transmitted to any external servers.

3. Personal Information Collected

1. Items: List of installed apps.
2. Purpose: To allow users to select apps for automation.
3. Retention: Data is loaded temporarily and is not stored separately.
